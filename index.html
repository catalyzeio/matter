Matter 0.0.2

Catalyze product styleguide and framework

This guide outlines the architecture of Matter, a product styleguide and framework. In release 0.0.2, Matter mostly consists of basic styles and is intended to be heuristic in nature. Contribution at this early stage is highly encouraged.

How to get Matter Grid system Blocks Navigation Icons Typography Colors Notifications Forms

How to get Matter

You can get matter by either cloning this repo and using the matter.css file (link to the Github repo in the header) or you can copy and paste the link below into your document.

https://some-link-that-will-exist.com/matter-0.0.2.css

Grid system

This grid system is comprised of 12 columns. The max width of the grid in Matter is 900px with a min width of 755px. Changing this is encouraged within the product you are working on, not in the master _grid.scss file.
